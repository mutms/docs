Certifications is a plugin set for Moodle that allows organisations to define
certifications tied to [Programs](../programs/), track compliance through
certification periods, and automate recertification. It is designed to make
workforce compliance with industry standards and regulations straightforward
to manage. Fully open source under GPL 3.0 with no restrictions on commercial
use.

## Features

- **Certification periods** — tie certifications to designated programs for
  compliance tracking, with configurable validity periods and expiration dates
- **Assignment sources** — multiple sources for assigning certifications to
  users, including manual assignment and cohort-based automation
- **Recertification rules** — advanced rules to match organisational
  recertification needs, with a separate program for initial certification
  and recertification
- **Certification catalogue** — users can browse available certifications;
  will be replaced by a universal catalogue in a future release
- **My certifications page** — a profile page giving learners a view of their
  current and past certifications
- **My certifications block** — a dashboard block for quick access to
  certification status
- **Notifications** — configurable notifications including the ability for
  supervisors to receive copies of learner notifications
