---
title: Training credits
description: Overview of the MuTMS Training credits plugin for Moodle LMS.
---

The Training credits plugin enables educators to assign training values to courses and programs using
Training credits custom fields. Program completion criteria can then be configured to require users to
accumulate a specified number of credits, which they earn by completing courses or other programs.

:::note
Training credits can only be earned in courses where the user is already enrolled.
:::

Training credits are an optional companion to the [Programs](../programs/) plugin — they are used as a
completion criterion within program content item sets.
