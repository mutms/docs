---
title: My programs overview page
description: The My programs overview page plugin for MuTMS Programs.
---

Provides a dedicated **My programs** overview page accessible from the main
menu — the programs equivalent of Moodle's standard **My courses** page.
The page offers card, list, and detail views of a learner's allocated programs.

:::note
The **My programs** menu option can be removed by disabling the **My programs overview page** block
in **Site administration > Plugins > Blocks > Manage blocks**.
:::
