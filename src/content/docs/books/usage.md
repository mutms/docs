title: Interactive book usage
description: How to create and use Interactive books in Moodle LMS.
---

## Creating a book

Add an Interactive book activity to a course the same way as any other Moodle activity. Once created,
add chapters and sub-chapters to build the book structure.

## Adding content to chapters

Each chapter can contain one or more content items. When editing a chapter, use the **Add content**
button to insert items of different types:

- **HTML text** — standard Moodle HTML editor content
- **Unsafe raw HTML** — raw HTML markup, restricted to users with the corresponding capability
- **Markdown text** — GitHub Flavored Markdown, including relative file links for images and attachments
- **Solution disclosure button** — hides content behind a button that learners expand on demand

## Navigation and table of contents

The book displays a chapter navigation sidebar and a full-page table of contents overview. Learners
can move between chapters using the navigation or return to the overview page at any time.

## Hidden content

Content items within a chapter can be individually hidden from students without hiding the entire
chapter, replacing the hidden chapters approach from the standard Book module.
