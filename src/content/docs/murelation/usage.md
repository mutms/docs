---
title: Supervisors & teams usage
description: How to configure and use the MuTMS Supervisors & teams plugin.
---

## Setting up a supervisors framework

1. Navigate to **Site administration > Users > User relation frameworks**.
2. Add a new framework using **Supervisors** mode.
3. Open a user's profile and use the **Actions** menu to assign a supervisor.

## Setting up a teams framework

1. Navigate to **Site administration > Users > User relation frameworks**.
2. Add a new framework using **Teams** mode.
3. Go to the **Teams** tab.
4. Add teams and assign team members.

Relationships are visible in user profiles.
