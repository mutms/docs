---
title: Custom home pages usage
description: How to configure and use the MuTMS Custom home pages plugin.
---

## Create a custom home page

1. Navigate to **Site administration > Appearance > Custom homepages > Home pages management**.
2. Press **Add page**.
3. Click the new page name to open it.
4. Turn on edit mode and add blocks to the page.
5. Turn off edit mode to preview.

## Primary menu integration

For each user, a list of visible pages is calculated and ordered by priority. The first page can be
configured to replace the standard **Home** item in the primary navigation menu. Remaining pages can
appear in a new dropdown alongside the **Home** item.

To replace the Home menu item:

1. Navigate to **Site administration > Appearance > Custom homepages > Home pages settings**.
2. Enable **Replace "Home" menu** and save.

To add a dropdown for additional home pages:

1. Navigate to **Site administration > Appearance > Custom homepages > Home pages settings**.
2. Enter a name in **Add additional home pages menu** and save.
